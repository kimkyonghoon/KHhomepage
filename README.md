주식시장에 대한 짧은 생각. 
1. 당신이 정말 수익을 얻을 자격이 있는지?
2. 하락장에서 패닉에 빠지지 않고 평정심을 유지하는지?
3. 상승장에서 포모에 빠지지 않고 평정심을 유지하는지?
4. 계속 질문해야 한다. 정말 여기가 꼭지 일까?
5. 계속 질문해야 한다. 정말 여기가 바닥일까?
6. 시장은 당신의 기업에 대한 평가나 전망을 항상 의심하고 질문한다.
   그리고 당신의 평가보다 때로는 더 낮은 가격을 때로는 더 높은 가격을 제시하곤 한다.
   당신을 믿어라. 당신의 평가를 믿어라. 그리고 때를 기다려라.
